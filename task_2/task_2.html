Тег code: призначений для вставлення програмного коду. Зазвичай вміст браузери відображають моноширним шрифтом
function add(a, b){ return a + b; }

Тег strong: застосовуєтьсяя для виділення жирним важливого тексту

Тег em: застосовуєтьсяя для виділення курсивом важливого тексту

Тег del: застосовується до тексту, що було видалено. Цей текст є у цій статі, а цей було видалено.

Тег ins: застосовується до тексту, що було додано. Цей текст є у цій статі, а цей було видалено, а цей текст додано.

Тег mark: виділяє текст. Виділений текст.

Тег q: позначає цитату. Станислав Лец утверждал: Чаще всего выход там, где был вход.

Тег samp: використовується для відображення результату роботи комп'ютерної програми.
